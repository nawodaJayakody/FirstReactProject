# FirstReactProject
Movie Search Web App

  A modern, responsive movie discovery application built with React.js. This project allows users to search for their favorite films, view trending releases, and     curate a personalized "Favorites" list that persists across browser sessions.

🚀 Features

  Real-time Movie Search: Instantly find movies using data from the TMDB API.

  Trending Movies: The homepage automatically populates with the most popular movies currently in theaters.

  Personal Favorites: Save movies to a local list using React Context for global state management.

  Persistent Storage: Your favorites are saved in the browser's localStorage, so they don't disappear when you refresh the page.

  Responsive Routing: Seamless navigation between the Home and Favorites pages using react-router-dom.

🛠️ Tech Stack

  Frontend: React 18+

  Build Tool: Vite (for fast development and bundling)

  Routing: React Router DOM

  Styling: Custom CSS with a focus on a "Netflix-style" grid layout

  API: The Movie Database (TMDB) API
